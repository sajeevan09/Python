Project Overview  
This is a simple console-based Tic-Tac-Toe game built with Python. Two players can play against each other, taking turns to place their marks (X and O) on a 3x3 grid. The game checks for a winner or tie automatically.

Tools  
Python 

Features  
Two-player gameplay (X vs O)  
Input validation to prevent invalid moves  
Automatically detects winner or tie  
Displays the board in a clear, easy-to-read format
