Project Overview  
The Expense Tracker is a simple Python program that allows users to track daily expenses. Users can add, view, summarize, and delete expenses stored in a CSV file. This project helps manage personal finances and practice basic Python programming.

Tools & Libraries  
Python  
csv – for storing and reading data  
datetime – for handling dates  
os – for file and folder operations

Features  
Add new expenses with date, category, amount, and optional note  
View all recorded expenses in a tabular format  
Summarize total and per-category spending  
Delete expenses by selecting their row number  
Automatic creation of CSV file in ~/Documents/ExpenseTracker/  

File Structure  
expenses.csv – automatically created to store expense records  
Python script – handles all operations (add, view, summarize, delete)  

How to Run
1. Make sure Python 3 is installed.
2. Save the Python file as expense_tracker.py.
3. Run the program in the terminal or command prompt:
4. python expense_tracker.py
5. Follow the menu options to manage your expenses.
