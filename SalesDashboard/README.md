Project Overview
This project is a web-based dashboard built with Streamlit that allows users to upload a CSV file of sales data and visualize it interactively. Users can preview data, see summary statistics, and plot trends for numeric columns over time.


Tools & Libraries
Python 
Streamlit – for building the interactive dashboard
pandas – for data handling and analysis
matplotlib – for plotting charts


Features
Upload a CSV file to view data dynamically
Preview the first few rows of the dataset
Display summary statistics (mean, min, max, etc.)
Select a numeric column and plot its trend
Automatic handling of date columns for proper time series plotting

Dataset
The dashboard works with any CSV file that contains numeric columns.
Numeric columns are used for plotting trends
Optional Date column for time-based plots

How to run:
1. Install required Libraries
2. Download and run dashboard.py
4. Open the link provided in your terminal to access the dashboard in your browser
5. Upload a CSV file and explore your sales data
